Install & Commands
---
```shell
# Commands

# If database support is required run the below command to install Sqlite database.
# All changes regarding database can be made under the 'config/' folder
# Create & Setup Database
npm run setup

# Run Tests
npm run test

# Run Standard Audit
npm run audit

# Run Bot
node dist/main.js
```
